    Wikibase 和 Wikidata 有什麼區別？

    Wikibase是一套用於管理鏈接開放數據的知識庫軟件，最初是為 Wikidata 項目編寫的，現在仍由 Wikidata 項目使用。

    Wikidata是 Wikibase 的最大實例，Wikibase 是一個任何人都可以編輯的免費知識庫。它的數據被維基百科、其姊妹項目以及任何其他想要使用大量開放通用數據的人使用。

    Method

    • 第一：從2019年12月到2021年1月，一名團隊成員收集現場筆記和數位制品（截屏、關聯數據、照片、模因）。
    • 第二：定性研究者用她從內容中推斷出的主題和觀點來標記數據。
    • 第三：第二位研究者進行獨立雙重編碼，獲得81.0%的注釋者間一致。最後，刘昂为檢查編碼後的數據，討論達成一致。
    • 第四：該研究只關注「公共場所」，
      • 利用推薦。研究了平台推薦系統推薦的社交媒體頁面和賬戶。

    Study Design

    • 第四：該研究只關注「公共場所」，使用四種方法來確定「公共場所」：
      • 新聞故事識別。使用新聞內容聚合器的搜索引擎來識別包含與標題中所選主題相關的關鍵字的新聞故事。接下來，觀察用戶對文章的評論。最後，在社交媒體上搜索提及這些新聞文章的帖子。在數據收集的早期階段和數據編碼的迭代過程中，關鍵字術語會不斷更新，直到形成一個綜合列表。
      • 頁面監控。該研究積極地監控特定的用戶、頁面和之前發現的臉書群組。
      • 跨頁討論跟蹤。該研究跟蹤帖子中的連結，這些連結指向不同網站上圍繞相同主題的討論，該研究將這些連結添加到跟蹤列表中。
      • 利用推薦。研究了平台推薦系統推薦的社交媒體頁面和賬戶。

    Study Design

    • 收集和存儲四種類型的資料：

      • 主題（Topics）是預先確定的。

      • 帖文（Postings）、網絡空間（Internet places）和意見（Opinions），來自質化研究。

    • ​意見是怎麼獲得的？

      • 帖文和網路空間是使用「公共場所」方法發現的數據。

      • 意見是描述這些數據的詞彙。意見被定義為用戶在帖子中表達的想法。

      • 質化研究和數據擴充階段構建新意見，並通過合並或拆分改變舊意見。

      • 因此，該研究在收集和標記數據的同時獲得了意見。

    Slide Classes in Editor UI

    You can use special syntax ("//= class-name") to specify custom class names that should be made available through the presentation editor interface. This makes it convenient to apply classes to specific slides. Here's an example class that blurs the slide that you apply it to:

    //= blurry-slide
.blurry-slide { filter: blur(5px); }

    Include fonts from Google or Typekit

    Click the T icon in the top right to load font families from Google Fonts or Typekit. Once you have added the fonts you want, don't forget to apply them to the content using CSS. For example, if you are loading the "Lobster" font family from Google Fonts you can apply it to all headings with: 

    .slides h1, .slides h2, .slides h3 {
  font-family: "Lobster";
}

    Include a custom font

    If you wish to include a custom font from another source you can point straight to the font file. 

    @font-face {
  font-family: "Cabin Sketch";
  src: url("https://static.slid.es/fonts/cabinsketch/cabinsketch-regular.woff") format("woff");
}

.slides h1, .slides h2, .slides h3 {
  font-family: "Cabin Sketch";
}
